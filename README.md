# Twitch Slot Bot

Простой слот-бот для Twitch чата. Команда `!slots` — три барабана, выигрыши и подсчёт очков.

## Установка

```bash
git clone https://github.com/твой_юзер/твой_репо.git
cd твой_репо
```

## Настройка

1. Скопируй `config.example.py` в `config.py`
2. Заполни свои данные:

| Поле | Описание |
|------|----------|
| `NICK` | Имя твич-аккаунта бота |
| `TOKEN` | OAuth токен (получить на https://twitchtokengenerator.com/) |
| `CHANNEL` | Канал для работы, с `#` (например `#мойканал`) |
| `OWNER` | Твой ник — у владельца нет 10-минутного кулдауна |

## Запуск

```bash
python3 twitch_slot_bot.py
```

## Команды

- `!slots` — крутить слоты

## Правила

- **Владелец** — кулдаун 3 сек
- **Все остальные** — кулдаун 10 мин
- Очки сохраняются в `slots_scores.json` между перезапусками
